# Video Downloader

A simple video downloader extension built using Node.js.

## Features
- Download videos easily
- Fast and simple
- Lightweight server

## Requirements
- Node.js
- npm

## Installation
```bash
npm install
```
## Run the Server
```bash
 node server.js
```

### Usage

1) Start the server.

2) Open your browser.

3) Go to Extensions.

4) Enable Developer Mode.

5) Click Load unpacked.

6) Select the extension folder.
